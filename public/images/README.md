# Image Template Guide

This directory contains images for the University Academic Portal. Follow this guide to add your own images.

## Directory Structure

```
images/
├── home/          # Home page images
├── courses/       # Courses page images
├── news/          # News page images
├── about/         # About page images
├── services/      # Services page images
└── support/       # Support page images
```

## Image Specifications

- **Format**: PNG or JPG
- **Recommended Size**: 1920×640 pixels (16:5.33 aspect ratio)
- **Minimum Size**: 1200×400 pixels
- **File Size**: Keep under 500KB for optimal loading
- **Quality**: Use high quality but optimize for web

## Tips

1. Use consistent styling across images
2. Ensure text overlays are readable
3. Optimize images before uploading
4. Use descriptive filenames
5. Maintain aspect ratio for best display

## Current Placeholders

The portal currently uses placeholder images. Simply replace the placeholder files with your own images using the same filenames, or update the image paths in the Blade templates.
