System Overview

What the portal manages

Student Experience

Registration, course enrolment, grades, fees, assignments, timetable, attendance, announcements, notifications, and messaging.

Teacher Workspace

Attendance recording, grade submission, assignment creation and marking, teacher announcements, and timetable access.

Staff Administration

Student and user management, courses, subjects, timetables, enrolment approvals, fee processing, grade review, attendance reports, and support inboxes.

Project Stack

Built as a real application, not a static prototype

  • Backend: Laravel 12 on PHP 8.2+
  • Frontend: Vue 3, Inertia.js, Vite, Tailwind CSS
  • Data: SQLite for local development, MySQL/MariaDB for production
  • Background processing: queued notifications plus scheduled attendance alerts
  • Extra capabilities: PDF export, image handling, Stripe payment flow, optional realtime messaging

Demo Note

Why this page is a showcase instead of the full app

GitHub Pages is ideal for public portfolio presentation, but this portal itself depends on server-side PHP, persistent storage, a SQL database, queue workers, and scheduled commands. Because of that, this github.io page is used as a clean project presentation layer.

The backend-ready deployment setup for the real application is included in the repository for Railway and Docker-based hosting.
